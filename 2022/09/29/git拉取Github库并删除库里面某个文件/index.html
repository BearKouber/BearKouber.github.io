git拉取Github库并删除库里面某个文件

1.拉取该库代码到本地

git拉取过程报错

$ git clone https://github.com/BearKouber/Web

SSL certificate problem: unable to get local issuer certificate

解决方法

git config –global http.sslVerify false

2.将远程分支与本地分支进行关联

  • 进入拉取到本地的文件夹

cd Web

  • 关联

git pull origin main

  • 查看是否连接成功

git branch -vv

可以看到master后面蓝色部分origin/main，说明master分支已经与origin/main绑定在一起了。

3.删除

git rm –cached config.js //删除config.js文件

git rm -r –cached config //删除config文件夹

4.推送到远程

git commit -m ‘删除某个文件’ //commit提交

git push //此处push时候弹出需要登录GitHub于是登录连接上了，应该是连接上才能push到github

于是刷新后，GitHub库里面的文件就不见了，耶✌
