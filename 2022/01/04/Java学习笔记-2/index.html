映射转换

  • 数据实体表设计 = 类的定义
  • 表中的字段 = 类的成员属性
  • 表的外键关联 = 引用关联
  • 表的一行记录 = 类的一个实例化对象
  • 表的多行记录 = 对象数组

​ 对于数据表与简单Java类之间的映射最好的解决步骤：先抛开所有的关联字段不看，写出类的基本组成，而后通过引用配置关联字段的关系
